# Overview

The Microchip SAMD21 Series Device Family Pack (DFP) is a [CMSIS-Pack](https://open-cmsis-pack.github.io/Open-CMSIS-Pack-Spec/main/html/index.html) that:

- Enables compatible tools with device support.
- Support files for the [MPLAB XC32 Compiler](https://www.microchip.com/en-us/tools-resources/develop/mplab-xc-compilers/xc32).
- Support files for the GCC compiler.
- Support files for the [Arm Compiler for Embedded](https://developer.arm.com/Tools%20and%20Software/Arm%20Compiler%20for%20Embedded).
- Support files for the [IAR Embedded Development Tools](https://www.iar.com/embedded-development-tools).
- Contains [System View Description [SVD](https://open-cmsis-pack.github.io/svd-spec/main/index.html) descriptions of the peripherals.
- Flash algorithms for the on-chip flash memory.
